### What is a Profile Identifier?
A Profile Identifier is information you want to store about the user. Typically, this will be an external wallet address 
or a social account.

We currently accommodate a variety of identifier types: Custom and External Wallets. Custom identifiers are versatile, allowing for the association of a specific 'service' name. This feature is particularly useful for a range of applications, such as internal player identifiers, third-party service integrations (like a Google identifier), or even for storing emails.


## Examples

### Adding external wallet identifers to a profile

Integrate wallet identifiers from our supported blockchain ecosystems directly into a user profile.


#### EVM
```TypeScript profileIdentifier.ts
import { StardustCustodialSDK } from '@stardust-gg/stardust-custodial-sdk';

const STARDUST_API_KEY = 'your-api-key';
const sdk = new StardustCustodialSDK(apiKey);
const profile = await sdk.getProfile(profileId); 
const userInputAddress = '<PLACEHOLDER>';
await profile.addEVMExternalWalletIdentifier(
    userInputAddress
);
```

#### SOL

```ts
await profile.addSOLExternalWalletIdentifier(
    userInputAddress
);
```
#### SUI
```ts
await profile.addSUIExternalWalletIdentifier(
    userInputAddress
);
```

### Adding custom identifers to a profile

Custom identifiers offer the flexibility to incorporate unique identifiers, tagged with a 'service' name, enhancing profile customization and integration capabilities.

```TypeScript profileIdentifier.ts
import { StardustCustodialSDK } from '@stardust-gg/stardust-custodial-sdk';

const STARDUST_API_KEY = 'your-api-key';
const sdk = new StardustCustodialSDK(apiKey);
const profile = await sdk.getProfile(profileId); 
const customServiceName = '<CUSTOM_SERVICE_NAME>';
const customId = '<PLACEHOLDER>';
await profile.addCustomIdentifier(customServiceName, customId);
```
